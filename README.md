# jinchen-fd

> 大厂全双工卷延迟、卷准确率；这个卷"像人"。

jinchen-fd 是一个从"人怎么聊天"出发的全双工原型：
- 🗣️ **半句抢答** — ASR 出 4 个字就开猜，不等你说完
- 😳 **猜错容错** — 猜歪了就"啊？不对不对…"，口吃+慌一下，反而真人感
- 🫠 **语气词过渡** — 卡壳时"嗯…""呃…"拖着，把思考过程露出来
- 🧠 **Archive 复用** — 直接搬 jinchen 的 SimHash 裁剪当记忆锚点
- 🎛️ **自动调参** — 用 feedback 日志跑 grid search，不用手动拧

Python 标准库即可跑，ASR/TTS/3D 是留给你自己换皮的接口层。
